Style's The Code You Wrap In Pre Tags

== Description ==

This plugin uses the Google Code Prettify Script to Highlight & Style Code

The plugin also adds styling to the existing pre Quick tag in your WordPress text editor

NO shortcodes which look bad if you stop using the plugin

All the code for this plugin comes from WordPress.org & Google

== Installation ==

1. Upload `plugin folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Thats it. 

Use the pre quicktag in the text editor to wrap your code
